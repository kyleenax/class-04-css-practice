# INFO 1300 (SP26) - Activity - CSS Practice

In this activity we will practice adding CSS to a project and creating a few element level styling rule sets.

This is a practice activity - not for credit. Feel free to expand as inspired.

Open this repository as a Codespace on GitHub.
Start the web server by clicking "Go Live" in the bottom right-hand corner.

## Instructions 

- This activity is a continuation, picks from, the class-03 activity.  Review the `styles/site.css` to see what we are starting with.

- View `index.html` in your browser using `Go Live` in the bottom right-hand corner.

- Add a generic font family to the entire page.

- Add a different generic font family to all the headers. 

- Add a rounded border to the main header `<h1>`.
  
- Add a left border to the `<block-quote>` elements.

- Explore other [properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Properties).
